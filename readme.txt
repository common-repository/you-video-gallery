You Video Gallery is a powerful but simple-to-use video gallery plugin for WordPress that gives you a ton of controls for automatically embedding videos from YouTube. You can control the embedding features with shortcode. There are different type of shortcode for different purpose. You can insert videos based on search, user, playlist, custom video with ID from YouTube.

"You Video Gallery" is a Responsive Youtube Video Gallery Plguin.

You can create Single, Fluid or Grid type Video Gallery easily.

Resize your video player size with your Page or Post body content. You can place shortcode any place in your WordPress Website. This is a fully Responsive plugin and adapt with all modern browser.

== Installation ==
Go to "Settings>Video Gallery Plugin", you'll see the Help Functionalities and helpful Documentation with Shortcodes.

e.g.

1. Upload 'you-gallery-plugin' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
